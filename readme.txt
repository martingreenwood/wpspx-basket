A slideout basket for WPSPX

== Description ==

This add-on creates a slide out basket for the WPSPX Plugin - Requires WPSPX

== Features out of the Box ==

- Floating basket icon
- Slideout panel with information, totals and link to the basket

== Installation ==

- Upload wpspx-basket to the /wp-content/plugins/ directory and activate
- Activate the plugin through the 'Plugins' menu in WordPress
- Visit the settings page under Settings > WPSPX or via the plugin page link
